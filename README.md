# 🎬 EditKaro

**EditKaro** is a modern, responsive landing page for a professional video editing and social media marketing agency. The website showcases creative services through an engaging UI featuring smooth animations, interactive effects, and a visually appealing design to enhance user experience.

---

## 🚀 Features

- 🎥 Modern Hero Section
- ✨ Smooth Animations & Transitions
- 🖱️ Interactive Custom Cursor
- 🌌 Animated Background Effects
- 📱 Fully Responsive Design
- 🎨 Creative UI/UX
- 📞 Contact Section
- ⚡ Fast & Lightweight
